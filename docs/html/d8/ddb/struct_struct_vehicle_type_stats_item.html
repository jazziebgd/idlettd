Detailed Description

[STRUCT] Vehicle type stats

Data structure used for vehicle stats of a particular vehicle type

Attention
This "struct" is not really used since Squirrel language has no structs as such. It's here solely for documentation purposes.

Member Data Documentation

◆ count

StructVehicleTypeStatsItem::count

Total number of vehicles of given type.

Value is for a particular vehicle type only, defined in StructVehicleTypeStatsItem::type

◆ idleBalance

StructVehicleTypeStatsItem::idleBalance

Total idle balance of all vehicles of given type in last year.

Value is for a particular vehicle type only, defined in StructVehicleTypeStatsItem::type. In practice it is just the StructVehicleTypeStatsItem::balance multiplied by idle multiplier setting.

◆ balance

StructVehicleTypeStatsItem::balance

Total balance of all vehicles of given type in last year.

Value (vehicles profit - vehicles cost) stored here is for a particular vehicle type only, defined in StructVehicleTypeStatsItem::type. Used to recalculate StructVehicleTypeStatsItem::idleBalance if needed.
