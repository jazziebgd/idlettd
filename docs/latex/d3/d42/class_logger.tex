\doxysubsection{Detailed Description}
Class that is used for all logging in Idle\+TTD. 

Class contains static methods for logging messages of various log levels. Checks game script setting \doxysectlink{index_config_sec}{log\+\_\+level}{2} to determine whether given log level is allowed.
